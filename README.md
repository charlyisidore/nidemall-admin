# NideMall (admin)

Admin frontend for [NideMall](https://github.com/charlyisidore/nidemall-server).

## Installation

```bash
yarn install
```

## Usage

```bash
yarn run dev
```

## Deployment

Create `.env.production.local` (or edit `.env.production`):

```
VUE_APP_BASE_API = 'https://example.com/admin'
```

Build:

```bash
yarn run build
```

The output is located in `dist/`.
